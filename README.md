# Rick and Morty App

Aplicativo desenvolvido com React Native e Expo que consome a API pública Rick and Morty para listar personagens da série. O projeto tem foco educacional, praticando consumo de API, navegação entre telas e exibição de dados em aplicações mobile.

## Tecnologias Utilizadas
- React Native
- Expo
- JavaScript
- API Rick and Morty

## Funcionalidades
- Listagem de personagens
- Visualização de detalhes do personagem
- Exibição de informações como nome, status, espécie e imagem
- Navegação entre telas

## Estrutura do Projeto
- App.js – Componente principal da aplicação
- Screens/ – Telas do aplicativo
- Components/ – Componentes reutilizáveis
- Services/ – Configuração de consumo da API

## Como Executar
1. Instale o Node.js
2. Instale o Expo CLI
3. Acesse a pasta do projeto
4. Execute `npm install`
5. Execute `npm start` ou `expo start`
6. Abra no emulador ou no aplicativo Expo Go

## Observações
Projeto desenvolvido para fins acadêmicos, com o objetivo de praticar consumo de API e desenvolvimento mobile com React Native.
